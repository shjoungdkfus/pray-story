## 짧은 설명 (최대 80자)

**후보 A (감성 중심)**
```
하나님이 오늘 나를 통해 써 내려가시는 나만의 기도 서신서. 매일의 기도와 응답을 기록하세요.
```
(48자)

**후보 B (기능 중심)**
```
매일 기도를 기록하고 응답을 확인하는 기도 일기. 통계·알림·묵상 커뮤니티까지 한 곳에서.
```
(45자)

> **최종 선택: B** (기능 중심)

## 자세한 설명 (최대 4000자)

```
📖 나의 서신서, PrayStory

PrayStory는 매일의 기도를 한 편의 편지처럼 써 내려가는 기도 일기 앱입니다.
"하나님이 오늘 나를 통해 써 내려가시는 양장 성경책" — 그 감성을 그대로 담아,
경건하고 따뜻한 화면 위에 당신의 기도와 응답을 기록합니다.

━━━━━━━━━━━━━━━━━━

✍️ 매일의 기도 기록
· 양면으로 펼쳐지는 성경책 감성의 메인 화면
· 좌우로 넘기며 날짜별 기도를 한눈에
· 원고지 줄이 그어진 작성 화면에서 정성껏 써 내려가는 손편지 감성
· 글자 크기 조절로 눈이 편안하게

🙏 기도 응답을 확인하는 기쁨
· 응답된 기도를 체크하며 하나님의 신실하심을 기억
· "이번 달 기록 N일", "연속 기록 N일" 통계로 꾸준함을 응원
· 월간·주간 달력에서 기도한 날을 한눈에

⏰ 기도 알림
· 매일 정해진 시간에 기도 알람
· 여러 시간대 설정, 개별 켜고 끄기

💌 함께 기도하는 커뮤니티
· 기도 그룹을 만들거나 초대로 참여
· 전체·그룹 편지 피드에서 서로의 기도를 나눔
· "소중한 이에게" 익명 편지로 마음을 전하기

🌙 나에게 맞는 화면
· 라이트 / 다크 테마 지원
· 따뜻한 크림빛 라이트, 깊은 심야 묵상의 다크

🔐 간편한 시작
· 카카오 · 구글 계정으로 3초 만에 시작
· 이메일 회원가입도 지원

━━━━━━━━━━━━━━━━━━

PrayStory와 함께라면, 흘려보내던 기도의 순간들이
당신만의 서신서로 차곡차곡 쌓여갑니다.
오늘의 기도를 써 내려가 보세요.
```
(약 620자 — 4000자 이내. 필요 시 사용자 후기/성경구절 등으로 확장 가능)
